# 生意参谋表格复制

一键复制淘宝生意参谋（sycm.taobao.com）表格数据为 Tab 分隔格式，直接粘贴到 Excel/Google Sheets。

## 功能

- **一键复制** — 点击按钮或快捷键 `Ctrl+Shift+X` 复制当前页表格
- **字段选择** — 独立开关控制每个字段是否复制（相关搜索词、搜索人气、各项变化率等）
- **表头开关** — 可选是否包含列标题行
- **操作列过滤** — 自动排除"操作"列
- **固定列过滤** — 自动排除左右固定列（重复列）
- **中文无乱码** — UTF-8 BOM 格式，Excel 直接粘贴正常显示

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `sycm-table-copy` 文件夹

## 使用

1. 打开生意参谋相关搜索词页面
2. **方式A** — 点击扩展图标 → 点击「复制当前页表格」
3. **方式B** — 直接按 `Ctrl+Shift+X`（Mac 为 `Command+Shift+X`）
4. 打开 Excel/Google Sheets → `Ctrl+V` 粘贴

## 设置

点击弹窗中的「字段设置」按钮进入设置页面：

- **包含表头** — 开启则复制带列标题行，关闭则只复制数据
- **数据字段** — 每个字段独立开关，可自由勾选要复制的列

## 文件结构

```
sycm-table-copy/
├── manifest.json     # Manifest V3 配置
├── popup.html        # 弹窗界面
├── popup.js          # 弹窗逻辑
├── content.js        # 核心：表格检测、数据提取、复制
├── options.html      # 设置页面
├── options.js        # 设置逻辑
└── icons/            # 扩展图标
```

## 权限说明

| 权限 | 用途 |
|------|------|
| `activeTab` | 与当前页面通信 |
| `clipboardWrite` | 写入剪贴板 |
| `storage` | 保存用户设置 |
| `*://sycm.taobao.com/*` | 仅在生意参谋域名运行 |

## 输出格式

- Tab 分隔（`\t`），可直接粘贴到 Excel
- UTF-8 with BOM，兼容中文
- 范围值（如 `600 ~ 1200`）原样保留
- 变化率（如 `+20.31%`）原样保留
